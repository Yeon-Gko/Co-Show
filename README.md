git branch

### 현제 브랜치 위치 정보 확인 
```javascript
git branch
```
### 모든 브랜치 정보 업데이트하기
```
git fetch --all
```

### 원하는 브랜치로 전환 
```
git checkout [브랜치명] 
```

### main 브랜치 최신 버전 가져오기 
- 새로운 브랜치를 생성하기 전에 항상 main 브랜치를 최신 버전으로 업데이트해야 합니다
- 이는 코드 충돌을 방지하고 최신 기능/수정사항을 포함하기 위함입니다
```
git pull origin main
```
